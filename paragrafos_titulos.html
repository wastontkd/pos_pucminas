HTML - Paragrafos e Titulos.

São vários elementos que permitem organizar sua pagina html. São eles:

Elementos

  1. Titulos = <h1>...<h6>
  2. Paragrafos = <p>
  3. Quegra de linha = <br>
  4. Voz Alternativa (Italico) = <i>
  5. Palavra Chave (Negrito) = <b>
  6. Importancia = <strong>
  7. Codigo Fonte = <code>
  8. Texto pre-formatado = <pre>
  9. Citacoes = <blockquote>

No exemplo acima foram usados quase todos os elementos para formatar o texto. Criei as ordens usando uma lista ordenada usando o elemento <ol>.
